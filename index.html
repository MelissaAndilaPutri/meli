proyek

  • Membuat atau mendesain ulang templat halaman web responsif
  • Situs web responsif adalah situs web yang dapat berfungsi di berbagai perangkat dan ukuran layar, baik di browser perangkat seluler maupun desktop. Jenis perangkat baru yang beraneka ragam saat ini (laptop, ponsel cerdas, tablet) membuat daya tanggap situs web juga menjadi lebih penting dari sebelumnya. Kamu bisa pertimbangkan untuk menggunakan template situs web dengan elemen desain yang telah ditentukan sebelumnya, yang akan memungkinkan untuk menerapkan desain responsif ke template yang ada, tanpa perlu membuatnya sendiri dari awal.
